# News Management App

A simple news management system (Flask + SQLite) with full CRUD and Tailwind CSS for styling.

Quick start

1. Create and activate a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Initialize the database (or visit `/initdb` after starting the app):

```bash
python app.py
# then open http://127.0.0.1:5000/initdb
```

4. Run the app:

```bash
python app.py
```

Open http://127.0.0.1:5000
